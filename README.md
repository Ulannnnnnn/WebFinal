## Project Description
This project is a **Weather Forecast Website** with authentication and user-specific data.
The backend is built with **Node.js, Express, and MongoDB**, and the frontend is a basic HTML/CSS/JavaScript interface.

Users can:
- register and log in
- manage their profile
- save favorite locations (cities)
- get weather forecasts for saved locations using an external Weather API

The project demonstrates REST API design, authentication with JWT, database integration, and basic frontend-backend interaction.

---

## Technologies Used

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (JSON Web Token)
- bcrypt (password hashing)
- Joi (validation)
- Axios (external API requests)

### Frontend
- HTML
- CSS
- Vanilla JavaScript (Fetch API)
